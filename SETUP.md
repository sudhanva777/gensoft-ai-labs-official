# Quick Setup Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Verify Logo File**
   - Make sure `public/Gemini_Generated_Image_quter9quter9qute.png` exists
   - The logo is used in Navbar and Footer components

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Open Browser**
   - Navigate to http://localhost:3000

## Important Notes

- The logo file must be in the `public/` folder
- All pages are fully responsive
